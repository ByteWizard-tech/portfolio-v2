# Omesh Sengar — Portfolio

A modern, high-performance **3D developer portfolio** built with **React**, **TypeScript**, **Three.js**, **GSAP**, and **WebGL**.

> 🌐 Live: [omeshsengar.com](https://omeshsengar.com)

---

## ✨ Highlights

- **3D / WebGL experience** powered by **Three.js** with an interactive character
- Smooth scroll animations with **GSAP** + **Lenis**
- Horizontal scrolling **project showcase**
- Career & experience timeline
- Tech stack pyramid with **42+ tools**
- Built-in chess game + AI chat on the `/play` page
- Fully responsive — desktop & mobile
- Deployed on **Cloudflare Pages**

---

## 🧰 Tech Stack

- **React 18 + TypeScript**
- **Three.js / React Three Fiber**
- **GSAP + ScrollTrigger**
- **Lenis** (smooth scroll)
- **Vite** (build tool)
- **HTML / CSS**

## 🚀 Getting Started

### 1. Clone

```bash
git clone https://github.com/ByteWizard-tech/portfolio-v2.git
cd portfolio-v2
```

### 2. Install

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

---

## 🧩 Customize

All personal data lives in one file — [`src/config.ts`](./src/config.ts):

- **Name, title, about text**
- **Projects list** (title, category, tech, GitHub link, description)
- **Career / experience** entries
- **Skills** (WhatIDo section)
- **Contact info** (email, GitHub, LinkedIn)

Change that file and everything updates automatically.

---

## 📄 Resume

Drop your resume PDF as `public/resume.pdf` — the RESUME button in the sidebar will link to it automatically.
